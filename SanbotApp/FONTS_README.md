# Font Files Required

This project uses the Poppins font family from Google Fonts. You need to download and add the font files manually.

## Required Font Files

Download the following font files from Google Fonts (https://fonts.google.com/specimen/Poppins):

1. `poppins_regular.ttf` - Regular (400)
2. `poppins_medium.ttf` - Medium (500)
3. `poppins_semibold.ttf` - SemiBold (600)
4. `poppins_bold.ttf` - Bold (700)

## Installation Steps

1. Download the Poppins font family from Google Fonts
2. Rename the files to match the expected names:
   - `Poppins-Regular.ttf` -> `poppins_regular.ttf`
   - `Poppins-Medium.ttf` -> `poppins_medium.ttf`
   - `Poppins-SemiBold.ttf` -> `poppins_semibold.ttf`
   - `Poppins-Bold.ttf` -> `poppins_bold.ttf`
3. Place the renamed files in: `app/src/main/res/font/`

## Alternative: Use Android Studio Font Download

In Android Studio, you can also:
1. Right-click on the `res/font` folder
2. Select "New" -> "Font Resource File"
3. Search for "Poppins" and download directly

The font files are not included in this repository due to licensing considerations.
